##PROJECT SUMMARY

Synkronos Mission Console is a web-based interactive application that simulates a complete recruitment and task assignment workflow. 
It allows users to create applicants by entering their details and completing a mandatory quiz, which evaluates their decision-making and skills.
The system then uses a decision engine to calculate a “Team Fit Score” based on factors like quiz performance, reliability, commits,
and task contribution, and recommends the best candidate.
A dashboard provides a detailed comparison of all applicants, while a task panel enables assigning and committing tasks to the selected candidate. 
Overall, the project demonstrates a structured, real-world hiring and evaluation system using dynamic UI, state management, and logical scoring mechanisms.
